# EcoGift Chile — Proyecto final consolidado (ES)

Este ZIP **sustituye** a los archivos intermedios anteriores.

Incluye en un solo proyecto:
- sitio en español
- catálogo poblado con categorías y productos visibles del sitio público
- trabajos/casos visibles
- clientes visibles
- soluciones
- flujo de cotización
- búsqueda y filtrado básico
- arquitectura ampliada
- vercel.json
- tsconfig.json
- utilidades SEO
- utilidades de búsqueda y filtro
- scaffolding de Sanity CMS
- estilos y componentes consolidados

## Ejecutar

npm install
npm run dev

## Deploy
Compatible con GitHub + Vercel.

## Rutas incluidas

/                Inicio
/productos       Catálogo con búsqueda y filtro
/productos/[slug] Detalle de producto
/trabajos        Portafolio
/trabajos/[slug] Detalle de caso
/soluciones      Casos de uso
/clientes        Clientes visibles
/cotizar         Formulario de cotización
